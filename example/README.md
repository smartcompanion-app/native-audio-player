# Example app

An example app to test the `@smartcompanion/native-audio-player` plugin.

## Getting started

 - `npm i` install required dependencies
 - `npm start` start example app for local development in the browser
 - `npm build` build project 
 - `npx cap sync` sync dist folder to android/ios projects
 - `npx cap open android` open android project inside android studio for testing
 - `npx cap open ios` open ios project inside xcode for testing
